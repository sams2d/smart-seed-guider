🌾 Smart Crop Guider

A Streamlit-based machine learning application that helps farmers identify the type of soil from an uploaded image and recommends the best crops to cultivate for that soil type.

Developed by: Sam Sundar
📧 Email: sundarsam805@gmail.com

🚀 Overview

This project uses a Convolutional Neural Network (CNN) model built with TensorFlow to classify soil types based on images. Once the soil type is identified, the system provides crop recommendations that are most suitable for that type of soil.

The main goal is to assist farmers and agricultural experts in optimizing crop selection and improving agricultural productivity using computer vision.

🧱 Features

  • 🧠 AI-based soil type detection (5 classes: Black, Cinder, Peat, Yellow, Laterite)
  • 🌱 Crop recommendation for each soil type
  • 🖼️ Easy image upload via Streamlit web interface
  • 💾 Local training and storage of model
  • 🧩 Modular code (separate scripts for training, app, and dependencies)

⚙️ Steps to Run

Step 1️⃣ — Install necessary dependencies

Before training or running the app, make sure you have all required libraries installed:

pip install streamlit tensorflow keras numpy pandas matplotlib opencv-python pillow scipy

Step 2️⃣ — Train the model

Make sure your dataset is properly structured. Then run:

python train_model.py

Step 3️⃣ — Launch the app

streamlit run app.py

🌾 How It Works

  • Upload an image of soil from your farm.
  • The trained CNN model classifies the soil type.
  • The system displays the predicted soil category.
  • The app recommends the best crops for that soil.
